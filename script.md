# Uzoza Launch Motion Script

## Purpose
Create a premium motion-graphics launch video that makes people immediately want to visit `uzoza.xyz`.

This piece should feel like it came from the real Uzoza product, not a generic AI ad.
It should use the same visual language, emotional tone, and aesthetic world already present on the site.

---

## Brand Alignment

### Pull directly from the actual Uzoza site
- Brand name: `Uzoza`
- URL CTA: `uzoza.xyz`
- Core promise: honest beauty intelligence, not flattery
- Emotional tone: premium, mysterious, sharp, aspirational, seductive, precise

### Visual language from the live product
- Background: deep black-brown luxury tones, not flat black
- Accent color: restrained metallic gold, not bright yellow
- Contrast: warm ivory text on dark surfaces
- Typography:
  - Headlines: `Playfair Display`
  - Supporting text / labels: `Josefin Sans`
- UI feel: minimal, expensive, editorial, scientific
- Motion: smooth, intentional, not chaotic, not "tech startup explainer"

### Use real project imagery where possible
Use the existing portrait assets from the site so the video feels unmistakably native to Uzoza:
- `/public/Nigerian_man_with_golden_ratio_analysis_lines.png`
- `/public/Brazilian_woman_with_symmetry_nodes.png`
- `/public/Korean_man_with_measurement_grid_and_score_readouts.png`
- `/public/South_Asian_woman_with_landmark_mesh_and_partial_score.png`
- `/public/Arab_man_diptych_Week_1_vs_Week_6_with_delta_score.png`
- `/public/Split_portrait_Black_man_and_European_woman.png`
- `/public/editorial-portrait.svg`

Do not use random stock imagery unless it matches this exact world.

---

## Creative Direction

This should feel:
- cinematic
- seductive
- luxurious
- credible
- slightly intimidating in a good way
- sharp enough to make viewers stop scrolling

This should not feel:
- cheesy
- overly corporate
- generic AI-generated
- overloaded with too many effects
- cartoonish
- like a crypto scam ad

The first 3 seconds must hook.
The final 5 seconds must sell the click.

---

## Motion Design Rules

- Use slow tension at the start, then accelerate with confidence.
- Let the gold accents glow subtly, never aggressively.
- Use elegant parallax, mesh reveals, score lock-ins, line tracing, UI float-ins, and leaderboard flips.
- Facial geometry overlays should feel real and precise.
- Keep spacing clean and editorial.
- Every frame should feel art-directed.
- Avoid fake hologram clutter.
- Avoid overusing lens flares, particles, and cheap glitch transitions.

---

## Color Direction

Base palette should reflect the site:
- Deep background: near-black brown
- Surface layer: rich charcoal / espresso
- Text primary: soft ivory
- Text secondary: muted warm gray
- Accent: muted gold / champagne gold
- Highlight: brighter gold only for emphasis moments

Suggested feeling:
- `#0D0A07`
- `#120D09`
- `#F6EAD9`
- `#DDCDB5`
- `#C9A96E`

Do not introduce unrelated neon colors.

---

## Typography Direction

- Use `Playfair Display` for major lines, score moments, and emotional reveal text.
- Use `Josefin Sans` for labels, technical overlays, dimension names, CTA support text, and UI metadata.
- Keep tracking slightly wide for labels.
- Let headline text breathe.
- Do not use default sans serif stacks if the motion tool allows custom fonts.

## Delivery Notes for Voice Generation

- Avoid exaggerated ad energy.
- Read it like luxury fashion meets precision technology.
- Keep pauses after the strongest lines:
  - "Your face has a score."
  - "Not flattery. Not filters. Just the truth."
  - "Discover your score at Uzoza dot X Y Z."
- Do not rush the URL.
- Make "Uzoza" sound deliberate and memorable.

## Editor Notes
